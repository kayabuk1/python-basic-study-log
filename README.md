# IoT Frontend Practice 💻

千葉県立船橋テクノスクール IoTシステム科でのフロントエンド学習（HTML / CSS / JavaScript）の記録と、取り組んだ実践課題をまとめたワークスペースです。

## 📁 ディレクトリ構成 (Directory Structure)
学習内容や課題ごとにフォルダを分けて管理しています。

- **`practice-introduce-webpage/`**
  - 基礎学習としてゼロから作成した自己紹介サイト（※記載情報は学習用のダミーです）。
  - HTMLの基本構造と、CSSのレイアウト（ボックスモデル、Flexbox/Grid）の基礎を実装しています。

- **`puzzleStudyJavaScript/`**
  - JavaScriptによる「15パズル（スライドパズル）」の実装と、非同期処理の学習コード。
  - DOM操作や、配列・乱数を用いたアルゴリズム（パズルのシャッフル処理など）の実装について学んでいます。

- **`vocational-school-exam/`**
  - テクノスクールでの試験関連のファイル群。

## 🛠 使用技術 (Technologies)
- HTML5
- CSS3
- JavaScript (Vanilla JS, 非同期処理など)

## 📝 学習の工夫 (Learning Approach)
**コメントを活用した学習の記録**
「ただコードを書いて動かす」だけでなく、「なぜそのコードで動くのか」を後から見返せるよう、ソースコード内に自分なりの解釈や調べた内容を細かくコメントとして残しています。
エラーのデバッグ過程や、理解に時間がかかったポイントもドキュメントとしてコード内に記述することで、知識の定着を図りながら学習を進めています。
