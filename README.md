# python-basic-study-log

## 目的
千葉県立船橋テクノスクールにおける、Pythonプログラミングの学習記録およびコードの実行ログを保存するリポジトリです。

## 学習のアプローチ
「プログラムが裏側でどのように解釈・実行されているか」を仕組みとして理解することを心がけています。
日々の学習ではNotebookLMを活用してPythonの内部挙動について対話した内容をコードのコメントとして書き残しています。

## リポジトリの構成と内容

当リポジトリには、大きく分けて以下の2つの領域における学習ログが含まれています。

### 1. Pythonの内部挙動に関する学習記録（ルートディレクトリ）
Pythonの言語仕様やオブジェクト指向の裏側の仕組みについて学習し、コメントとして記録したコード群です。
* **オブジェクト指向の深層と継承**: `class`・`type`・`object` の関係性（メタクラスと始祖）、継承のメカニズム、特殊メソッド検索のバイパス機構の学習。
* **属性アクセスの裏側とカプセル化**: `__dict__`（インスタンス辞書）の働き、`__setattr__` や `__getattribute__` の挙動、データディスクリプタ（`__get__`、`__set__`）を用いたカプセル化の確認。
* **実行モデルと関数の動的生成**: `__call__` を用いたインスタンスの関数化、スコープ（LEGBルール）、クロージャ、デコレータの内部機構の確認。

### 2. Raspberry Piを用いた組み込み制御（`RaspberryPi_study/`）
Raspberry Piの実機を用いた、ハードウェア制御および各種センサーとの通信ログです。
* **GPIO制御**: LEDの点灯・消灯（`01_02_led.py`）、スイッチ入力の検知（`01_03_sw.py`）。
* **センサー連携**: 温湿度センサー（DHT20）等を用いたI2C通信のデータ取得（`06_03_dht20.py`, `dht20_smbus.py`）。
* **マルチメディア制御**: カメラモジュールを用いた画像・動画撮影やOpenCVによる処理（`03_01_camera.py`, `opencv_practice`）、音声ファイルの再生（`03_02_mp3.py`）。

## 実行環境
* 言語: Python 3.x
* 主な環境: 船橋テクノスクール実習環境 / Raspberry Pi
